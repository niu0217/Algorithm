# 英文输入法

## 题目

[87、英文输入法](https://blog.csdn.net/m0_47384542/article/details/131877947)

## 代码

### 示例

```c++
输入
The furthest distance in the world, Is not between life and death, But when I stand in front of you, Yet you don't know that I loveyou.
f

输出
front furthest

说明：

从用户已输入英文语句The furthestdistance in the world, Is not between life and death, But when I stand in frontof you, Yet you dont know that 1
love you."中提炼出的单词，符合“f作为前缀的， 有"furthest"和"front", 按字典序排序并在单词间添加空格后输出，结果为"front furthest"。
```

### 代码

```c++
#include<iostream>
#include<algorithm>
#include<set>
#include<cctype>
#include<vector>
#include<string>
#include<sstream>

using namespace std;

int main()
{
    string s;
    getline(cin, s);
    istringstream iss(s);
    vector<string> strVec;
    string word;
    while(iss>>word) {
        strVec.push_back(word);
    }

    //规范每个单词
    vector<string> standStrVec;
    for(string word : strVec) {
        string clean = "";
        for(int i = 0; i < word.size(); i++) {
            if(isalpha(word[i])) {
                clean.push_back(word[i]);
            }
            else {
                standStrVec.push_back(clean);
                clean = "";
            }
        }
        standStrVec.push_back(clean);
    }

    //寻找匹配的前缀
    string preStr;
    cin>>preStr;
    set<string> result;
    for(string word : standStrVec) {
        bool isPre = true;
        if(preStr.size() > word.size()) {
            isPre = false;
        }
        else {
            for(int i = 0; i < preStr.size(); i++) {
                if(word[i] != preStr[i]) {
                    isPre = false;
                }
            }
        }
        if(isPre) {
            result.insert(word);
        }
    }

    if(result.size() == 0) {
        cout<<preStr<<endl;
    }
    else {
        for(string word : result) {
            cout<<word<<" ";
        }
        cout<<endl;
    }
}
```
