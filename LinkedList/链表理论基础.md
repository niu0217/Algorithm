# 链表理论基础

链表节点在内存中是不连续分布的。

链表节点的定义：

```c++
// 单链表
struct ListNode {
    int val;  // 节点上存储的元素
    ListNode *next;  // 指向下一个节点的指针
    ListNode(int x) : val(x), next(NULL) {}  // 节点的构造函数
};
```

链表和数组的对比：

+ 数组：
  + 插入/删除：O（n）；
  + 查找：O（1）；
  + 适用场景：查找频繁，较少删除；
+ 链表：
  + 插入/删除：O（1）；
  + 查找：O（n）；
  + 适用场景：较少查询，频繁删除/插入；
